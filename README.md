# MBLookProjectBugs Tracker
Bug treking for MBLook library project

Welcome to the bug tracker for **mblook.net**. This repository is used to report, track, and resolve issues related to the site.
## About
This repository serves as a public bug tracker for the mblook.net project. Whether you're a user reporting a bug or a contributor helping to fix it, you're welcome to participate! The goal is to improve the platform's stability and user experience.

## How to Report a Bug
To report an issue or suggest an enhancement:
1. Go to the **[Issues tab](https://github.com/niufir/MBLookProjectBugs/issues)**.
2. Click **"New Issue"**.
3. Select the **"Bug Report"** template (or "Feature Request" if applicable).
4. Fill out the form with:
   - A clear title (e.g., "[Bug]: Search not working").
   - Steps to reproduce the issue.
   - Expected vs. actual behavior.
   - Screenshots (if available).
   - Environment details (OS, browser, version).
5. Submit the issue. Add relevant labels (e.g., `bug`, `enhancement`) if needed.
